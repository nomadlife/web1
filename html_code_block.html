Compared: TEXTAREA, XMP, PRE, SAMP, CODE

Note that CSS can be used to override default fixed space fonts in each or all these.

XMP: Example text with Newlines, tabs & space, html tags etc <b>displayed natively</b>. However, note that &amp; (&) will not act as an escape char.. Eg: &lt;u&gt;(text)&lt;/u&gt; 
PRE: Example text with Newlines,
tabs & space,
  html tags etc are interpreted, not displayed.
    However, note that & still acts as an escape char..
      Eg: <u>(text)</u>
SAMP: Example text with Newlines, tabs & space, html tags etc are interpreted, not displayed. However, note that & still acts as an escape char.. Eg: <u>(text)</u> CODE: Example text with Newlines, tabs & space, html tags etc are interpreted, not displayed. However, note that & still acts as an escape char.. Eg: <u>(text)</u>
